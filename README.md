# Weather data check
Consulta el clima de la ciudad seleccionada.

  - Selecciona la ciudad
  - Lista los datos

## Da un vistazo al app operativa 🚀

_Deployment Domain [Netlify](https://upbeat-austin-63cc721.netlify.app/)_


### Instalación 🔧

_Instalación_

```
NPM install
```

_Desarrollo_

```
NPM run start
```

_Producción_

```
NPM run build
```
